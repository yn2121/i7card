# i7card manager

アイドリッシュセブンのカード所持状況を管理する**非公式**Webツール。

- 公開URL: https://i7card.vercel.app/
- GitHub: https://github.com/yn2121/i7card
- 管理者X: @sy_yss2

---

## 構成概要

```
フロントエンド（index.html）
  └─ GitHub Pages / Vercel で静的配信

バックエンド（GAS）
  ├─ カードマスター読み込み（Googleスプレッドシート）
  └─ 所持データ読み書き（別スプレッドシート）

認証
  └─ Google Identity Services（OAuth）
```

## Googleスプレッドシート構成

### カードマスター（GASがバインドされているスプシ）

| 列 | ヘッダー名 | 内容 |
|---|---|---|
| A | カードID | 数値ID（連番） |
| B | カード名 | カードのフルネーム |
| C | 名前 | キャラクター名 |
| D | レア | N / R / SR / SSR / UR |
| E | 覚醒先ID | 覚醒先カードのID（なければ空） |
| F | メモ | 任意メモ |
| H | （抜け番チェック用） | `=IF(A4-A3>1,"↑間が空いています","")` ツールに影響なし |

### 所持データスプシ（`MY_DATA_SS_ID`）

| 列 | 内容 |
|---|---|
| A | GoogleユーザーID |
| B | 所持データJSON `{"001":"list","002":"bench",...}` |

---

## 所持ステータスの種類

| キー | 表示名 | 意味 |
|---|---|---|
| `list` | 一覧 | 手元にあって一覧に入れているカード |
| `awake` | 覚醒/お別れ済 | 覚醒・お別れしたカード |
| `bench` | 控え室 | 持っているが一覧に入れていないカード |
| `viewed` | 未所持閲覧済 | 持っていないが確認済み |
| `unseen` | 未所持未閲覧 | デフォルト（未設定と同じ扱い） |

## 主な機能

- カード一覧表示（無限スクロール、60件ずつ）
- レアリティ・キャラフィルター（複数選択可）
- カード名・キャラ名テキスト検索
- ID昇順/降順ソート
- 所持ステータス変更（自動保存、デバウンス1.2秒）
- ステータス変更後2秒間は画面からカードが消えない（誤操作対策）
- スキルレベル記録
- 覚醒元・覚醒先モーダル表示（クリックで関連カードポップアップ）
- GALLERY PROGRESS（所持状況の進捗バー）
- 貸せるカード・借りたいカード画像出力（20件/枚、PNG）
- Googleログイン（データ同期）/ ゲストモード（閲覧のみ）

---

## セキュリティ

- GAS側でリクエスト元オリジンをチェック（`ALLOWED_ORIGINS`）
- Googleログインは Google Identity Services で完結（パスワード等はサイトに渡らない）
- キャラクター画像等の著作物は一切使用していない（テキスト情報のみ）

---

## スケール感

| 基盤 | 目安ユーザー数 |
|---|---|
| GAS（現在） | 100〜150人程度 |
| Firebase（移行候補） | 1,000〜2,000人程度 |

移行時はGoogleユーザーIDをキーにデータをそのまま引き継げる。
